# 📦 منظومة إدارة المخازن والفواتير الذكية | Warehouse & Invoice Management System

تطبيق ويب متكامل واحترافي مصمم لإدارة المخازن، المنتجات، والعمليات الحسابية والفواتير بدقة وسهولة متناهية. يتميز النظام بالربط السحابي الكامل، الحماية الفائقة ضد أي محاولات تلاعب أو اختراقات، مع تدعيم عمليات البحث والفرز البصري الذكي.

---

## ✨ الميزات الرئيسية (Key Features)

### 1. 🛡️ أمان وحماية فتاكة (Security Shield)
* **حظر أدوات المطورين (Anti-DevTools):** جدار حماية يمنع فحص العناصر (Inspect Elements) أو التلاعب بالأكواد في لوحة التحكم (Developer Console).
* **حماية من ثغرات حقن برمجيات المواقع (XSS Protection):** تصفية تلقائية وفورية لأي مدخلات مشبوهة أو أكواد خبيثة لمنع ثغرات الحقن.
* **حظر تلويث البروتوتايب (Prototype Pollution Protection):** تأمين كائنات الجافاسكريبت لضمان تشغيل آمن ومستمر.
* **تعطيل وظائف eval والدوال الديناميكية:** منع تنفيذ جافاسكريبت عشوائية محقونة في صفحات العرض.

### 2. 📊 إدارة سريعة للمخازن والمنتجات (Inventory Dashboard)
* **تنظيم مستويات المخزون:** لوحة تحكم تعرض كميات البضائع والمنتجات، المنصرف والمتاح بالتفصيل.
* **إصدار الفواتير الآلي:** واجهة تفاعلية لتوليد فواتير الشراء والبيع والطباعة الفورية.
* **التفويض والربط السحابي:** تخزين وإدارة بيانات المنتجات والمبيعات سحابياً لحظة بلحظة عبر Firebase Firestore.

### 3. ☁️ اتصال سحابي آمن (Firebase Cloud Sync)
* **المزامنة اللحظية:** تحديث دائم لحركة المخازن لجميع الأجهزة النشطة في نفس الوقت.
* **تسجيل دخول آمن:** نظام صلاحيات متكامل لمنع تداخل صلاحيات الموظفين مع الإدارة.

## 🚀 كيفية التثبيت والتشغيل المحلي (Setup & Running Locally)

لتشغيل المشروع محلياً، تأكد من تثبيت [Node.js](https://nodejs.org/) ثم اتبع الخطوات التالية:

```bash
# 1. تثبيت الحزم البرمجية والاعتمادات
npm install

# 2. تشغيل بيئة التطوير المحلية
npm run dev

# 3. بناء نسخة الإنتاج الجاهزة للنشر
npm run build
```

## 🌐 النشر على GitHub Pages ومزامنة السحاب

المشروع مُهيأ للعمل تلقائياً على استضافات الـ Static Hosting مثل GitHub Pages مع تفعيل نظام المزامنة السحابية. لربط استضافتك بـ Firebase بشكل سليم:
1. اذهب لمشروعك في **Firebase Console**.
2. انتقل إلى قسم **Authentication** ثم لعلامة تبويب **Settings / Authorized Domains**.
3. أضف نطاق استضافتك على جيت هاب (مثال: `creativ-1020.github.io`).

## 🛠️ التقنيات المستخدمة (Tech Stack)

* **React (TypeScript)** لبناء الواجهات الرسومية التفاعلية بكتابة كود قوية ومقاومة للأخطاء.
* **Vite** لتوفير أفضل أداء وسرعة في البناء والتصيير.
* **Tailwind CSS** للتصميم العصري المتجاوب بالكامل مع الشاشات المختلفة.
* **Firebase (Authentication & Firestore)** لقواعد البيانات اللحظية والتحقق من هوية المستخدمين.
* **GitHub Actions** للرفع والبناء التلقائي للمستودع عبر الويب.
